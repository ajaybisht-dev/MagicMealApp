# 🍔 MagicMeal - AI Powered Food Ordering App

MagicMeal is an intelligent food ordering application that leverages AI to provide personalized meal recommendations and a seamless ordering experience. The app focuses on smart suggestions, real-time interactions, and a smooth user interface.

---

## ✨ Features

- 🤖 AI-powered food recommendations  
- 🍽️ Browse menu and order food  
- 💬 Chat-based food assistant (LLM integration)  
- 🎯 Personalized suggestions based on user preferences  
- ⚡ Fast and responsive UI  
- 📦 Real-time order handling
